# 📦 cobra-md

Generate README.md documentation from a Cobra CLI project

cobra-md reads the cmd/ folder of a Cobra project and generates a single README.md documenting all commands, flags, and examples.

---

## 📚 Commands

- `version`

## `version`

Print the version number of the application

Print the version number of the application.

### 📋 Usage

```bash
cobra-md version
```
