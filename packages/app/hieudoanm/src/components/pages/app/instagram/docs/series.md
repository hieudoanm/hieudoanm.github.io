## 1. How It Works

### Goal

Explain how something functions.

### Core Question

> How does this actually work?

### Best For

- Technologies
- Biology
- Neuroscience
- AI
- Software
- Engineering

### Structure

1. Hook
2. Why it matters
3. Components
4. Process
5. Example
6. Misconceptions
7. Summary

#### Examples

- How LLMs Work
- How Sleep Works
- How Redis Works
- How GPS Works
